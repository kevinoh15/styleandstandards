This style guide has been added to help aid you in the development of new features and pages across Chatham's various products. The hope for the guidelines here is help make building pages faster along with creating a consistent client experience for all users of our products.

This site has only been formatted for viewing in Google Chrome and though you may use pieces of HTML and CSS from here, it is recommended you still do a thorough job of cross-browser testing any pieces you use. Valentina Sottano and George Frank have been the developers overseeing the implementation of these ideas and should be considered for any guidance you may need with the HTML or CSS.

As this site is considered to be a living document, it is encouraged that you send any recommendations for it's improvement to either Brett Whitham, Valentina Sottano, or George Frank.
